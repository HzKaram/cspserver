Reporting API demo

Web page with violations, that uses the new reporting API (v1).

The following reports will be automatically generated by loading this page:
  • Permission policy violation (see script.js)
  • Document policy violation (see script.js)
  • Deprecation (see script.js)
  • CSP violation
  • COEP violation (see below)
To generate a crash report, type chrome://crash/ in your browser from this page.
